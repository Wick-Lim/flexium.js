# Flexism App

Realtime-first Fullstack Framework built with [Flexium](https://github.com/user/flexium).

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
