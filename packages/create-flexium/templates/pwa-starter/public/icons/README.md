# PWA Icons

Replace these placeholder icons with your own app icons.

## Required Sizes

Generate icons in the following sizes:
- 72x72
- 96x96
- 128x128
- 144x144
- 152x152
- 192x192
- 384x384
- 512x512

## Icon Generator Tools

- [PWA Asset Generator](https://github.com/nickel-dev/pwa-asset-generator)
- [Real Favicon Generator](https://realfavicongenerator.net/)
- [Maskable.app](https://maskable.app/) for maskable icons
