# Flexium DevTools

Browser DevTools extension for debugging Flexium.js applications.

> **Note**: This is a private package and is not published to npm.

## Features

- Inspect reactive state and signals
- View component tree
- Track state changes in real-time
- Performance monitoring

## Development

```bash
# Build the extension
npm run build

# Watch mode
npm run dev
```

## Installation

1. Build the extension: `npm run build`
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select the `dist` folder

## Usage

1. Open Chrome DevTools (F12)
2. Find the "Flexium" panel
3. Inspect your application's reactive state
